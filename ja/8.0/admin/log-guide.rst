==========================
ログファイルのダウンロード
==========================

概要
====

ここでは、 |Fess| 
で出力されるログファイルのダウンロードについて説明します。

設定方法
========

表示方法
--------

管理者アカウントでログイン後、メニューのログファイルをクリックします。

|image0|

ダウンロード
------------

表示されているログファイル名をクリックすると、ログファイルをダウンロードすることができます。

.. |image0| image:: /images/ja/8.0/admin/log-1.png
